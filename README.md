# Sistema-aluguel-carro
Sistema de aluguel de carros desenvolvido em Java

DESCRICÃO:
O sistema permite calcular o valor total de um aluguel de carro com base no modelo escolhido, quantidade de dias e quilômetros percorridos.

- Cálculo do valor da diária
- Cálculo baseado nos dias alugados
- Cálculo adicional por km rodado
- Exibição do valor total a pagar

O valor total é calculado somando:
 (valor da diária × quantidade de dias)
 (quantidade de km × valor por km)

  Tecnologias utilizadas
- Java
- Eclipse IDE
